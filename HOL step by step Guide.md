- Visual Studio Code を起動 (デスクトップ上の準備されたショートカットをダブルクリック)

- Git の初期設定を実行

  - ユーザー名の設定

    ```
    git config --global user.name "User Name"
    ```

    ※ User Name を自身の名前に変更

    ```
    git config --global user.email "Email@Address"

    ```
    git config --list --global
